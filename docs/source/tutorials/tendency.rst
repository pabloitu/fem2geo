Guide to input scripts
----
Examples are provided for Normal, Stirke slip and Reverse faults in a rectangular crustal domain containing a pre-existing weak fault zone.

Following the first example file provided in tutorials/a_dilation_NormF, one
must:

1) identify the path to the model run .vtu result file

2) Define the domain area within which the stress field components are located in a set of meshed elements. It can be a sphere or a paralleleliped.

3) define a set of pre-existing fractures.

4) define the output directory and file names
